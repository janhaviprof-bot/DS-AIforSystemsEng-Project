# App description — News for People in a Hurry

**News for People in a Hurry** is a Shiny for Python dashboard that pulls live headlines from the **New York Times Top Stories API**, ranks and labels them (for example breaking and trending signals), and presents them as a fast, card-based feed. Readers can filter by category tabs (All, Business, Arts, Technology, World, Politics), narrow the time window, optionally filter by AI-assigned sentiment, and choose a summary tone so short blurbs match how they want to read. The goal is to give busy people a credible snapshot of what matters *right now* without opening dozens of tabs.

The app’s primary external data API is the **NYT Top Stories** endpoint (`/svc/topstories/v2/{section}.json`), queried in parallel across several sections and merged into one deduplicated dataset. **OpenAI** (Chat Completions, model configured in `config.py`) powers the “soft” intelligence layer: per-article sentiment and impact labels for articles in the visible time window, tone-controlled summaries on the cards, section-level briefs, and the multi-agent “Signal Studio” narrative. Market context comes from **Yahoo Finance** via the `yfinance` library, both for the workflow’s market snapshot and for optional quote tools in the research assistant.

Beyond a simple news reader, the team added **agentic orchestration** and **tool calling** in two places. First, **Signal Studio** runs a staged pipeline: lightweight section packets are assembled from the current feed, then three coordinated agents run in sequence—cross-section linking, world mood and market stance from the news, and market validation that compares that stance to live tape. Agent 3 can invoke an OpenAI function tool **`get_market_snapshot`** so the model chooses which symbols to refresh before returning structured JSON; the server still prefetches a default snapshot for the UI and for fallback. Second, each card can trigger **Dive deeper**, which opens a **research brief** produced by an OpenAI tool loop that may call **`wikipedia_lookup`** (English Wikipedia search + summary extract) and **`yahoo_finance_quote`** (ticker snapshot) before writing a concise, attributed brief. That pattern is retrieval-augmented in spirit: the model grounds answers in fetched text rather than guessing facts.

These features create value for different stakeholders. **Casual readers** get scannable cards, filters, and quick AI summaries so they can stay informed in minutes. **Students and analysts** benefit from Signal Studio’s cross-section synthesis and the explicit news-versus-market check, which encourages critical thinking about whether headlines and prices tell the same story. **Course reviewers and teammates** can see a concrete systems design: clear agent roles, tool boundaries, caching, and progressive loading so the page stays responsive while heavier work finishes in the background.
